# External Libraries and Tools

This page lists the major external tools used in the repository and links to their official documentation.

## Angular

- Purpose: frontend framework for the visualization app.
- Official docs: [angular.dev](https://angular.dev)

## CMake

- Purpose: build system for the Fortran engine.
- Official docs: [cmake.org/documentation](https://cmake.org/documentation/)

## Doxygen

- Purpose: reference documentation for Fortran source files.
- Official docs: [doxygen.nl/manual](https://www.doxygen.nl/manual/)

## MkDocs Material

- Purpose: GitHub Pages documentation site.
- Official docs: [squidfunk.github.io/mkdocs-material](https://squidfunk.github.io/mkdocs-material/)

## Playwright

- Purpose: browser-based end-to-end tests for the Angular app.
- Official docs: [playwright.dev/docs/intro](https://playwright.dev/docs/intro)

## Vitest

- Purpose: frontend unit testing and coverage.
- Official docs: [vitest.dev/guide](https://vitest.dev/guide/)

## Wiki.js

- Purpose: self-hosted wiki software for the beginner-first knowledge base.
- Official docs: [js.wiki](https://js.wiki/)

## Weather provider documentation

### Open-Meteo

- Purpose: free forecast data source.
- Official docs: [open-meteo.com/en/docs](https://open-meteo.com/en/docs)

### National Weather Service API

- Purpose: United States hourly forecast data source.
- Official docs: [weather.gov/documentation/services-web-api](https://www.weather.gov/documentation/services-web-api)

### MET Norway Locationforecast

- Purpose: global forecast data source.
- Official docs: [api.met.no/weatherapi/locationforecast/2.0/documentation](https://api.met.no/weatherapi/locationforecast/2.0/documentation)

### 7Timer

- Purpose: additional free forecast source.
- Official docs: [7timer.info/doc.php?lang=en](https://www.7timer.info/doc.php?lang=en)
